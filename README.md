Foody App

Foody App est une application web de gestion et de consultation de
recettes culinaires. Elle permet aux utilisateurs de découvrir des recettes
et à un administrateur de gérer le contenu.

1-Technologies utilisées

1.1_Frontend
React.js (Vite)
React Router
Axios
CSS

1.2_Backend
Node.js
Express.js
MySQL
Multer (upload d’images)

2_Installation et exécution locale
Prérequis
Node.js (v16 ou plus)
npm
MySQL
Git

3-Cloner le projet
git clone https://github.com/JamilaAssila/foody-app.git
cd foody-app

4-Lancer le frontend
npm install
npm run dev

L’application sera accessible sur : http://localhost:5173

5-Lancer le backend

Dans le dossier backend :
npm install
npm start

L’API sera accessible sur : http://localhost:5000

Comptes de test

Administrateur
Email : admin@foody.com
Mot de passe : 123
Accès au Dashboard : http://localhost:5173/admin
Utilisateur

6-Fonctionnalités

--Utilisateur

Consulter les recettes
Rechercher et filtrer les recettes
Voir les détails d’une recette

--Administrateur

Ajouter une recette
Modifier une recette
Supprimer une recette
Upload d’images

7-Tester l’application

Ouvrir http://localhost:5173
Se connecter en tant qu’admin
Accéder au Dashboard
Ajouter, modifier ou supprimer des recettes
Tester la recherche côté utilisateur
